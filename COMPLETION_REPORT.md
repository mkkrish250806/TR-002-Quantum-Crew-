### 🔄 **Integration Flow**

```
User Types Message
    ↓
Frontend validates input
    ↓
API Service adds headers
    ↓
Sends to Backend (http://127.0.0.1:8000/chat)
    ↓
Backend processes (RAG + LLM)
    ↓
Returns response with metadata
    ↓
Frontend displays message
    ↓
Shows confidence, intent, urgency
    ↓
Updates metrics dashboard
    ↓
Persists in session storage
```
